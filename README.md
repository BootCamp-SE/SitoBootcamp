# Sito Bootcamp

## Version

    - 1.0.0 

## Description

Il sito del bootcamp è sviluppato in Node js (18.7.0) e Espress.js (4.18.1), è pensato per dare una vetri verso l'esterno al bootcamp oltre ad aggiungere una gestione interna 

## How to run

```bash
$ npm i
$ node app.js
```
or

```bash
$ npm i && npm run start
```
